5 Задание:
1) Организовать цикл перебора, с нуля до нужного числа.
2) Перевод в двоичную систему счисления (bin(), f'{N:B}').
3) Проверяем на условие дописывам и заменяем нужное.
4) Переводим в десячиную и проверяем на условия.

```python
for N in range(516):
    b = f'{N:b}' // bin(N)
    if N%2 == 0:
        b+='10'
    else:
        b = '1'+b+'10'
    
    if int(b,2) >516:
        print(b)
        break;
```
